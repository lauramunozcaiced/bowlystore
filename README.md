## Key Features

Bowly Store is a bowls sales portfolio project  that simulates the delivery process with a Map Interface using <a href="https://leafletjs.com/" target="_blank">Leaflet.js</a>. Implemented local storage to save deliveries and show tracking deliveries information


## How To Use

To clone and run this application, you'll need [Git](https://git-scm.com). From your command line:

```bash
# Clone this repository
$ git clone https://github.com/lauramunozcaiced/bowlystore.git

$ cd bowlystore

```
